Prereques on RPi

1. Slideshow Application - https://github.com/NautiluX/slide
2. Unclutter - Hide Cursor - apt-get install unclutter
3. NodeJS/NPM - https://github.com/nodesource/distributions 
4. SMB support/etc
5. Chromium Browser - apt-get install chromium-browser
6. xdotool - automation - apt-get install xdotool
